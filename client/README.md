# Portfolio Profesional - Maximiliano Jiménez

Portafolio de alto rendimiento construido con **React**, **HeroUI v2.8+** y **Tailwind CSS**.

## 🚀 Características

-   **Aesthetica Glassmorphism**: Efectos de desenfoque y transparencias modernas.
-   **Modo Oscuro/Claro**: Switch funcional con `next-themes` (modo oscuro por defecto).
-   **Galería Dinámica**: Proyectos cargados desde `db.ts` con visor de detalles cinemático.
-   **Animaciones Premium**: Transiciones staggered con `framer-motion`.
-   **Formulario de Contacto**: Validado con `react-hook-form` e integrado para envío a `majf2704@gmail.com`.

## 🛠️ Tecnologías

-   **Frontend**: React (Vite) + TypeScript
-   **UI**: HeroUI (NextUI) + Tailwind CSS
-   **Icons**: Lucide React
-   **Animations**: Framer Motion

## 📦 Instalación y Uso

1.  Asegúrate de tener **Node.js** instalado.
2.  Instala las dependencias:
    ```bash
    npm install
    ```
3.  Inicia el servidor de desarrollo:
    ```bash
    npm run dev
    ```

## 📂 Estructura del Proyecto

-   `src/data/db.ts`: Base de datos de proyectos.
-   `src/components/`: Componentes modulares de la interfaz.
-   `src/types.ts`: Tipados de TypeScript para el proyecto.
-   `public/images/`: Assets, fotos y multimedia.
